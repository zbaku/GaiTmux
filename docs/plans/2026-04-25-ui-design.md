# AIShell UI Design - Material 3 深色主题

> Created: 2026-04-25
> Status: Draft

## 设计原则

- **终端优先**：全屏终端为主界面
- **AI 辅助**：悬浮按钮唤起 AI 对话
- **Material 3**：现代 Android 设计语言
- **深色主题**：适合长时间使用

---

## 1. 主界面布局

### 全屏终端 + 会话 Tab

```
┌─────────────────────────────────────────┐
│ [会话1] [会话2] [会话3] [+]            │  ← 顶部会话 Tab
├─────────────────────────────────────────┤
│                                         │
│  $ adb devices                          │
│  List of devices attached               │
│  ABC123    device                       │
│                                         │
│  $ fastboot flash boot boot.img         │
│  Sending 'boot' OKAY [0.5s]             │
│  Writing 'boot'  OKAY [0.3s]            │
│  Finished. Total time: 0.800s           │
│                                         │
│  $ _                                    │
│                                         │
│                               ┌─────┐   │
│                               │ 🤖  │   │  ← 悬浮 FAB
│                               └─────┘   │
└─────────────────────────────────────────┘
```

### 会话 Tab 栏

**功能**：
- 显示当前打开的会话列表
- 点击切换会话
- `+` 按钮新建会话
- 长按关闭/重命名会话

**样式**：
- Material 3 SecondaryTabRow
- 可横向滚动
- 深色背景 + 主题色选中态

### 悬浮 FAB（Floating Action Button）

**位置**：右下角

**样式**：
- Material 3 FAB
- 圆形，直径 56dp
- 主题色背景 + 白色图标

**交互**：
| 操作 | 行为 |
|------|------|
| 点击 | 弹出 AI 对话窗口 |
| 长按 | 进入语音输入模式 |

**滚动行为**：
- 终端向上滚动时自动隐藏
- 滚动停止后延迟显示

## 2. AI 对话弹窗

### Bottom Sheet 风格

**点击 FAB 弹出**：

```
┌─────────────────────────────────────────┐
│  🤖 AI 助手                    [─] [✕] │  ← 标题栏
├─────────────────────────────────────────┤
│                                         │
│  用户: 帮我把 Download 里的图片整理一下 │
│                                         │
│  AI: 好的，我来帮你执行：               │
│  $ find /sdcard/Download -name "*.jpg"  │
│  找到 23 个 jpg 文件                    │
│                                         │
│  $ mkdir -p /sdcard/Download/图片整理   │
│  已创建目录                             │
│                                         │
│  ⚠️ 需要确认: 移动 23 个文件            │
│  [确认] [取消]                          │
│                                         │
├─────────────────────────────────────────┤
│  [🎤] 输入指令或描述任务...        [→]  │  ← 输入栏
└─────────────────────────────────────────┘
```

### 弹窗特性

| 特性 | 说明 |
|------|------|
| 样式 | Bottom Sheet，从底部滑出 |
| 高度 | 初始 50% 屏幕高度，可拖动调整 |
| 手势 | 下拉关闭，上拉展开 |
| 键盘 | 输入时自动调整高度 |

### 输入栏

**组件**：
- 左侧：语音输入按钮 `[🎤]`
- 中间：文本输入框
- 右侧：发送按钮 `[→]`

**语音输入**：
- 点击麦克风开始录音
- 录音时显示波形动画
- 松手自动识别并发送

### 消息类型

| 类型 | 样式 |
|------|------|
| 用户消息 | 右对齐，主题色气泡 |
| AI 文本 | 左对齐，灰色气泡 |
| 命令执行 | 代码块样式，等宽字体 |
| 确认请求 | 警告色边框 + 操作按钮 |

## 4. 页面结构总览

| 页面 | 入口 | 功能 |
|------|------|------|
| 终端主页 | 默认启动 | 全屏终端 + 会话 Tab |
| AI 弹窗 | FAB 点击 | AI 对话交互 |
| 设置页 | Tab 栏右侧 | 配置管理 |
| 会话详情 | Tab 长按 | 会话管理 |

### 导航流程

```
启动 App
    │
    ▼
终端主页 ──────┬──→ [FAB] ──→ AI 弹窗
              │
              ├──→ [Tab+] ──→ 新建会话
              │
              └──→ [Tab长按] ──→ 会话管理
                              │
                              ├── 重命名
                              ├── 关闭
                              └── 清空历史
```

object TerminalColors {

## 7. 动画规范

| 组件 | 动画 | 时长 |
|------|------|------|
| FAB 显示/隐藏 | 淡入淡出 + 缩放 | 200ms |
| Bottom Sheet 展开 | 从底部滑入 | 300ms |
| Tab 切换 | 滑动过渡 | 150ms |
| 按钮点击 | 波纹效果 | 100ms |
| 语音波形 | 脉冲动画 | 循环 |
